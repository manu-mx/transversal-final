Bienvenido(a):

En este sitio se abordarán los temas y recursos necesarios para la comunidad, aquí se pueden observar los avaces y los resultados del proyecto transversal que se ha trabajado en este ciclo 2022-B al 2023-A.

Sobre el proyecto

Fase 1 En esta fase esta la etapa inicial donde se propone iniciar la estructura de la página web.

Fase 2 En esta fase es la etapa de desarrollo sobre la página web, donde se integran los recursos obtenidos en parciales pasados, para tener más diseño en el sitio web.

Fase 3 En esta fase es la etapa final donde se espera tener listo y terminado este proyecto, con todos los procesos y objetivos alcanzados, realizado de la manera esperada y correcta.

Origen. Diversos problemas de escazes de agua en la zona, y reutilizar las aguas residuales de los climas para diferentes usos.

Sobre el proyecto se tiene que completar cada fase o proceso para que este trabajo en conjunto se complemente de cada una de las partes y así lograr el objetivo deseado. La página web y el sistema hidraulico hacia los baños escolares.
